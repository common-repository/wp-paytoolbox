WP PayToolbox is a great eCommerce plugin to present content from your powerful Paytoolbox shop on WP page.

== Description ==
WP PayToolbox is a free eCommerce plugin to present content from your PayToolbox eCommerce platform on the WP page. With this plugin you can present categories and products with images, prices, descriptions etc. Checkout is done on the PayToolbox platform.

= PayToolbox =
Paytoolbox is a complete eCommerce platform to sell your digital products with downloads and support tickets. Each product can be configured with limited time subscription and support. Powerfull back admin panel offers large amount of features including: time, product based promotions with percentage or fixed price discount. Product associations and much more! 

= Sell Anywhere =
Place Quick Sell links anywhere, even on social media or on email. Quick Sell link can target shopping cart or checkout process directly.  

= Support Tickets =
Paytoolbox offers advanced support ticketing system just in case you would like to e.g. sale software with support option. 

= Payment Gateways =
Multiple payment gateways supported like PayPal, Stripe, PayU or Bank Transfer.

== Installation ==

= Minimum Requirements =

* PHP version 5.2.4 or greater (PHP 5.6 or greater is recommended)

= Installation =

1. Upload and activate the plugin
2. Update BASE_URL and credentials constants in wppaytoolbox.php
3. Create a standard Wodpress page or post where you want to show a shop homepage (list of categories)
4. Add shortcode [wpptb-shop] to the page and save it.
5. Open the post URL and it should show you all categories and subcategories from your Paytoolbox shop

= Updating =

Automatic updates should work like a charm; as always though, ensure you backup your site just in case.
